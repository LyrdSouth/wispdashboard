Features

Everything you need for your Discord server

Moderation Tools

Manage your server with powerful moderation commands including ban, kick, timeout, and more. All commands include role hierarchy checks and DM notifications.

  • ?ban / /ban Ban users with custom reasons
  • ?timeout / /timeout Temporarily mute members
  • ?kick / /kick Remove users from your server

Image Manipulation

Transform images with various effects and manipulations. Upload an image with your command to apply these effects instantly.

  • ?gif / /gif Convert images to GIF format
  • ?caption / /caption Add text captions to images
  • ?fry / /fry Apply a deep-fried effect to images
  • ?mirror / /mirror Mirror images horizontally

Utility Commands

Useful tools to enhance your server experience and customize the bot to your liking.

  • ?snipe / /snipe View the last deleted message
  • ?setprefix / /setprefix Customize the command prefix
  • ?setsecuritylog / /setsecuritylog Set up security logging

Slash Commands

All commands work with both traditional prefix commands and Discord's slash command system. Enjoy the best of both worlds!

  • Synchronized command descriptions
  • Intuitive parameter inputs
  • Autocomplete support for parameters
  • Works alongside prefix commands

CHANGELOG

v1.2.0

New Features

  • Slash command support for all moderation commands:
    • /ban Ban users with reason and DM notification
    • /kick Kick users with reason and DM notification
    • /timeout Timeout users with duration and reason
    • /untimeout Remove timeout from users
    • /unban Unban users by ID
  • Slash command support for utility commands:
    • /snipe Show last deleted message
    • /prefix Show current command prefix
    • /setprefix Change command prefix (admin only)
    • /ping Check bot latency
    • /help Show command list
    • /repo Get repository link
  • Auto-sync of slash commands on bot startup
  • Support for both prefix and slash commands simultaneously

Improvements

  • Updated help command to show both prefix and slash command syntax
  • Improved command descriptions for slash commands
  • Added ephemeral responses for sensitive commands
